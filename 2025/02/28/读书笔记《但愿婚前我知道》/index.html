读书笔记《但愿婚前我知道》

发表于 2025-02-28 分类于 读书笔记 阅读次数：
本文字数： 3.3k 阅读时长 ≈ 6 分钟

《但愿婚前我知道》

盖瑞·查普曼博士

46个笔记

◆ 点评

2023/8/3 认为好看

◆ 第一章 但愿婚前我知道 彼此恋慕并不足以成为幸福婚姻的基础

>> 那些见面时的激情、彼此相悦的默契、身体接触时的心动，就像圣代冰激凌上的红樱桃，让圣代冰激凌更加美味诱人。

◆ 第二章 但愿婚前我知道 浪漫要分两步走

>> 现在我知道。与第一阶段的浪漫相比，第二阶段特别需要恋人们有意识地去做一些事情。双方都要尽一些努力，让你们之前的感情过渡到另一个层次。

>> 我们习惯于使用自己的爱语，用自己喜欢被爱的方式去爱对方。但要注意，如果对方感受被爱的方式与你不一样（爱语不一样），那么他（她）往往就不会感受到你所付出的爱。

>> 1.肯定的言词：这种爱的语言是使用话语去肯定别人。

>> 2.服务的行动：对于使用这样爱语的人，行动比言语更能表达他们的爱。

>> 3.接受礼物：对于某些人而言，收到礼物时被爱的感觉最强烈。

>> 4.精心的时刻：这要求你全心全意地关注对方，如果你是以一边看电视、一边想着其他事的状态和某人待在一起，那就不是精心的时刻。

>> 5.身体的接触：我们早就知道身体的接触可以带来多大的感情力量。

◆ 第三章 但愿婚前我知道 “有其母必有其女”和“有其父必有其子”的说法是有道理的

>> 大多数人都要比自己想象的更像自己的父母。

>> 因为我们都是从小在父母身边长大的，因此不好的沟通方式很难被自己发觉，与此相反，我们还会认为这很正常。父母的交流方式潜移默化地影响着我们，直到别人家庭不同的交流方式引起了我们的注意，才使我们看到改变沟通方式的必要性。值得高兴的是，这些不好的沟通方式是可以改变的，你们约会期间就为这种改变提供了最好的时机。

>> 1.把你与你父亲的相似之处列一个清单，再把你看到的父亲性格中的缺点列出来。如果你的女友和你的父亲有足够多的接触，让她根据她的观察，也列出以上两个清单。

◆ 第四章 但愿婚前我知道 如何用不争执的方式处理分歧

>> 在那之前，没有人告诉我们，争执对每个家庭来说都是正常的，没有家庭不会遭遇冲突。原因很简单：因为我们都是不同的个体，所以我们的需要不同，爱好不同，甚至连讨厌的事也不同。

>> 首先，你必须接受一个事实：我们不得不去面对分歧。

◆ 第五章 但愿婚前我知道 道歉的力量

>> 没有道歉和原谅的婚姻是不健康的。

>> 这是因为妻子小时候，妈妈教育她道歉时该说：“我错了，你可以原谅我吗？”她此时正等待科尔能这样向她道歉，因为对妻子而言，说“对不起”并不意味着道歉。

>> 道歉的五种语言

以下依次简要介绍我们通过研究发现的五种道歉的语言：

1.表示悔意

>> 举例来说，你可以这样道歉：“对不起，我回家晚了一个小时，我知道你一直在等我去看电影，现在电影已经开始半个小时，你可能也不想再去看了。我觉得自己这样没有时间观念，只关心自己办公真的很糟糕，我除了自责谁也不能责怪，我知道这次我让你大失所望了。”

>> 承担责任

>> 愿意承担责任，并承认自己伤害他们的言行是错误的。

>> 3.弥补过失

这种道歉的语言寻求表达“补偿”的意愿

>> 4.真诚地表达改正行为的意愿

>> 5.请求原谅

>> 幸福的婚姻并不取决于两个人都完美，真正至关重要的是当你们伤害了对方之后，要学会道歉。

◆ 第六章 但愿婚前我知道 原谅不是靠一时的感觉

>> 原谅不是消除我们的记忆

>> 第二，原谅并不意味着伤害所带来的各种不好结果都能消除。

>> 第三，原谅并不一定能重塑信任的关系。

>> 第四，原谅并不总是能够带来和好。

◆ 第七章 但愿婚前我知道 家里的卫生间不会自己变干净

>> 让我问你一个个人问题，当你结婚以后，你认为谁会打扫家里的卫生间呢？

>> 那些忽视或否定父母的生活模式会对自己的婚姻产生影响的想法，确实很不现实。成熟的情侣会坦诚地交流各自的期望，遇到不同的观念时会一起商量，从而在婚前就达成他们对婚姻角色定位的共识。

>> 如果女孩子曾就读于很保守的教会大学，那她对女性在婚姻中的角色可能会有另一种完全不同的看法。男方也是一样，他的教育和信仰背景也会很深地影响他对人生的理解，决定他对未来婚姻角色的认识。

>> 如果你们非常认真地在考虑结婚，请你们各自把维持未来家庭所需做的事列一个清单，请注意，这份清单要细化到你们的用车、谁去购买、准备三餐、洗衣服、打扫地板等。然后对照你们两人列出的清单，彼此补充，综合起来形成一个包含你俩所想到的一切家庭相关事务的大清单。

◆ 第八章 但愿婚前我知道 如何精打细算用好钱

>> 如果我们在婚前确定了一个财务计划，就会因此免受很多不必要的烦恼。

>> 一个普遍的原则，是住房方面的开支不要超过你们净收入的40%。

同时，明智的购物也能够带来显著的差异

>> 游戏是这样做的：一个周五或是周六的晚上，你们一起去百货商店，走过那些货架，随便找一件吸引你眼球的商品，看看上面的标签，谈谈它是多么地招你喜欢。然后彼此说：“我们不必非要买它，难道这不好极了吗？”然后你会看到，当其他人怀里抱得满满地走向收银台签名付账时，你们却兴高采烈地手牵手走出商店，享受着你们不是建立在物质上的快乐。我强烈建议那些新婚夫妻都来做这个游戏。

◆ 第九章 但愿婚前我知道 夫妻双方获得性满足不是自然而然的事

>> 但结婚之后我才发现，给一方带来天堂般美妙体验的性，对另一方来说却好像地狱般恐怖。

>> 男人关注的是性爱，而女性关注的是关系。

>> 我后悔自己在婚前对性缺乏认识的第二点是，对妻子来说，性爱之前的爱抚比性爱本身更美好。

◆ 第十章 但愿婚前我知道 婚姻把各自的原生家庭连接在一起

>> 第一个需要引起你注意的是假期的安排。

>> 你以请求的方式进行协商的效果，会大大好于你以要求的方式进行协商。

>> “你们每次来我们都很高兴，而且我们也希望能多花时间和你们在一起，但是我想提一个请求，与其每次没有通知就经过我们这里，你们能不能考虑在来之前给我们打个电话，看看是否合适我们晚上的时间。我这么说，是因为上周四你们来，结果我不得不熬夜到凌晨才把报告赶完，第二天好上交，要是你们周五晚上来我就会方便很多。我这样建议有道理吗？你们是否能考虑一下？”

>> 如何与你们姻亲父母保持良好正面的关系，我最后还有一条建议，就是去了解并向他们表达“爱的语言”，当你的姻亲父母真正感到被爱时，就会为你们在一起的协商带来良好的氛围，没有什么比你用他们的爱语来关爱他们，能更深地表达对他们的爱意。

>> 6.当你的父母与你谈一些他们的看法，与你所持的观点不同时，你能做到暂时不发表自己的观点，而是先站在他们的立场上，倾听他们的想法，然后用智慧的方式回应他们吗？彼此分享你们在聆听父母意见的方面成功或失败的经历。

◆ 第十二章 但愿婚前我知道 有什么样的个性就有什么样的做事方式

>> 结婚以前，我梦想着和妻子每天能一起起床，共进早餐。但婚后才发现，凯若琳不是个早起的人，对于她来说吃不吃早饭根本不重要。回想约会的时候，她确实在电话中告诉过我：“不要在早晨给我打电话，我中午之前说的话都不算数。”

>> 以色列约旦河的水不断流入死海，死海中的水却从不外流，这一点和很多人的个性有相同之处。

>> 那些进取型的丈夫或妻子，相信每一天都是推进事业的新机遇，只要是他们想要的，认为正确的，觉得能做成的，都会不遗余力地去争取，克服一切困难去达到目标。还有一种被动型的人，他们把时间用在思考、分析、担忧各种“万一”的情况上，总是用等待的方式期望事情朝好的方向发展。他们的人生格言是：“愿意等候的人是不会缺少好处的。”
